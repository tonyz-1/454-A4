Setting up GLFW on Linux with command-line 'make'
-------------------------------------------------

1. Open a terminal window.

2. Change to this directory.

3. If GLFW is not installed on your system, type:

       sudo apt install libglfw3-dev



To compile and run the code
---------------------------

4. Type 'make'.

5. Run code as './demo1' or './demo2' or './demo3', etc.
